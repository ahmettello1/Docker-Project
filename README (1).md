# Docker Network Practice Lab

## Proje Amacı
Docker networking kavramlarını pratik etmek için tasarlanmış bir laboratuvar. Özel bridge network oluşturma, container bağlama ve multi-container uygulamaları kapsar.

## Açıklama
- Özel network'ler ile container'lar arası iletişim.
- Resource limitleri (CPU, memory).
- Uygulama + veritabanı entegrasyonu.

## Nasıl Çalıştırılır?
1. Repo'yu klonla: `git clone https://github.com/R4152-Ahmet/Docker-Network-Practice-Lab.git`
2. Klasöre gir: `cd Docker-Network-Practice-Lab`
3. Script'i çalıştır: `./Docker\ Network\ Practice\ Lab.sh` (Windows'ta Git Bash kullan)

## Gereksinimler
- Docker Desktop yüklü olsun.
